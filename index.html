CORGI

GPU Accelerated Fractal Engine

Corgi uses GPU compute shaders to calculate fractal images at extremely high speeds. It uses a perturbation formula to calculate the images with very high precision, allowing for extremely high levels of magnification. This image was rendered at a zoom level of 294, which is a 19,807,040,​628,566,​084,398,​385,987,584 times magnification. The Vulkan-like library wpgu for GPU compute and renders the UI with egui.

BLACKBOX

Modern Kernel-Level Process Tracer

Blackbox is a proof-of-concept eBPF-based process tracing and analysis program for Linux. It runs any other program with an eBPF probe attached that tracks every system call the program makes. These system call events are then collected and analyzed to build a higher-level model of the program's behavior.

  • eBPF
  • Rust
  • Linux

Ray-Graph is a 3D graph visualization web application. It uses dynamically-generated OpenGL shaders transpiled from user-defined LaTeX functions to render uncommonly structured functions that many other 3D graphing applications struggle with.
